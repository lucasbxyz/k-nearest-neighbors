# k-nearest-neighbors
[![Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/lucasbxyz/k-nearest-neighbors/HEAD)

This project applies the K Nearest Neighbors algorithm to classify data points into target classes based on multiple features. The dataset contains anonymized feature columns, and the goal is to find the optimal value of K using the elbow method and evaluate classification performance. This project is an assignment from the Udemy course: [Python für Data Science, Machine Learning & Visualization](https://www.udemy.com/course/python-data-science-machine-learning).
